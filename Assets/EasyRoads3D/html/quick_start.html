IMPORT PACKAGE

First import the EasyRoads3D v3 package. This will add the following new menu item in the Unity Editor:

Main Menu > GameObject > 3D Object> EasyRoads3D > New Road Network

A new game object "Road Network" will be created in the scene hierarchy and auto selected. In the Inspector you will see the following
EasyRoads3D v3 Inspector toolbar:

To create a new road, click the second tab from the left . Select one of the preset roads from the dropdown and and click the Add New Object button. A new game object will be created in Road Network / Road Objects and is automatically activated. Hold SHIFT and click in the scene to add markers. The road will appear after two markers are added. You can add more markers by holding SHIFT and click, you can insert markers by hitting the I key. You can change marker positions by moving the mouse to a marker, it will light up. Click this handle. You can select the traditional position handle or the rectangle move handle in the Inspector or by pressing the W key (Position handle) or M key (Move handle). The E key will display the rotation handle which can be used to add tilting to the road.

- Click here for additional road options in the scene
- Click here for more information about setting up road types.

Next we will create some crossings. Intersections can be added to the start or end of the road. Select the first or last marker of the road you just created. A list of available crossings prefabs will be displayed in the Inspector. Click the prefab button that you want to attach to the selected marker. It will automatically attach to the selected marker.

You will see red handles in the center of each crossing connection. Click one of Drag one of the red handles and drag the mouse to create a new road starting at that connection.

Alternatively you can place crossing prefabs directly in the scene. In the Inspector select the third tab from the left from the main EasyRoads3D Toolbar. Select the prefab you want to add to the scene, hold SHIFT and click in the scene where you want to position it.

You can move crossings by clicking on the yellow sphere handle in the center. Just like with markers you can move the crossing with the traditional position handle (W key) or the rectangle move handle (M key). Additionally you can also rotate the crossing, click the E key to show the rotation handle.

- Click here for additional crossing prefab options in the scene
- Click here for more information about setting up crossing prefab types

Next we will flatten the terrain to match the road shapes. Click the middle Mountain tab from the main EasyRoads3D Toolbar in the Inspector. You will see a couple of options to customize the final build. Click the Build Terrain to actually flatten the terrain to the road shapes. Click the Restore Terrain button to switch back to Edit Mode.

- Click here for additional options in Build Mode
- Click here for more information about setting up the preferred terrain handling and other terrain related options.
